# FigureDetection
 Программа определяющая тип нарисованной двумерной фигуры.

 Рисуешь примерно двумерную фигуру и программа определяет ее тип
окружность, эллипс, ромб, треугольник (в тч равнобедренный и прямоугольный) и тп.

Смотрите скриншоты.
